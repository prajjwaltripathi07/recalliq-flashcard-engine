# RecallIQ — Flashcard Engine

RecallIQ is an AI-powered flashcard engine that converts PDFs into high-quality, practice-ready flashcards for active recall and long-term retention.

## Features
- Upload educational PDFs
- Extract text from PDFs
- Generate 15–25 AI-powered flashcards using Claude
- Organize flashcards into decks
- Study in an interactive flashcard interface
- Rate recall as Easy / Medium / Hard
- Simple spaced repetition scheduling
- Dashboard with total decks, total cards, due today, and mastered cards

## Tech Stack
- Frontend: React, Vite, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB + Mongoose
- AI: Anthropic Claude API
